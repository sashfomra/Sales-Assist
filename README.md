# 📊 SalesIQ – Smart Coaching AI

An AI-powered sales call analysis platform that evaluates conversations and provides actionable coaching insights such as talk ratio, interruptions, and question quality.

---

## 🚀 Features

* 🎙️ Call Analysis (Audio Processing)
* 🧠 AI Coaching Feedback
* 📈 Metrics:

  * Talk Ratio (Rep vs Client)
  * Interruptions
  * Question Quality (Open vs Closed)
* 📊 Visualization using charts
* ⚡ Fast frontend powered by Vite + React

---

## 🛠️ Tech Stack

* Frontend: React.js (Vite)
* Charts: Chart.js / react-chartjs-2
* Linting: ESLint
* Package Manager: npm
